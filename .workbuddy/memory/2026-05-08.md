# 2026-05-08 工作记录

## 全场景职场生存智囊Agent 项目深度优化分析

完成了对项目的第三轮全面代码 review，梳理出 29 个优化点，分为：

### Bug 修复（P0）
1. FileOperationTool 无路径穿越防护（漏洞）
2. FileBasedChatMemory 静态 Kryo 非线程安全
3. AiChatAgent 注入了被注释掉的 pgVectorVectorStore Bean
4. OrchestratorAgent.chatStream 并非真正流式（子 Agent 同步）
5. TerminalOperationTool 硬编码 cmd.exe（Linux不兼容）

### 功能缺失
1. doChatWithRag/doChatWithTools/doChatWithMcp/doChatWithReport 4个方法没有 HTTP 接口
2. 首页 Home.vue 没有 LoveMaster 入口卡片（路由存在但无入口）
3. Manus 不支持 chatId 多轮对话
4. SessionManager 内存存储、重启丢失
5. 知识库没有 list/delete 接口

### 安全问题
1. api-key 明文在 application.yml
2. Token 明文暴露在 URL 中（EventSource 限制导致）

### 代码质量
1. ReReadingAdvisor 未被任何地方使用
2. 日志级别全局 DEBUG 可能泄露用户数据
3. HealthController 返回裸字符串不符合 Result 规范

用户询问还有哪些可以优化，已提供完整分析报告（29个优化点，按优先级 P0/P1/P2 排序）
