# 高情商博弈流：优雅说“不”的极限推拉艺术

## 一、 拒绝加班：不是不想加，是资源没对齐
### 1.1 日常非紧急加班
- **错误示范：** “我家里有事。”（太私人，容易被道德绑架）
- **高情商话术：** “领导，现在手头有A、B两件事。如果要紧急插队这个新活，A的截止时间能不能顺延到后天？或者咱们协调个人支援一下？”
- **落地做法：** 甩出排期图，让领导做选择题。
### 1.2 周末/假期夺命连环Call
- **场景：** 周末接到工作电话。
- **应对策略：** 非紧急响铃3声后接起：“喂领导，我在外面信号不太好。这事儿急吗？我大概两小时后能碰到电脑，如果特别紧急我先让家里把电脑开机，我电话指挥现场人员处理。”

## 二、 边界感建立：看似柔软，实则带刺
### 2.1 非职责范围的“求帮忙”
- **场景：** “帮我做个PPT呗，你审美好。”
- **高情商回话：** “我这会正被老板追着搞那个表格，焦头烂额的。不过我这儿有份以前用过的模板发你，你往里填内容绝对没问题，特别出活儿！”
- **策略：** 提供情绪价值+工具支持，但不提供劳动力。

## 三、 会议中的沉默螺旋破解
### 3.1 被问住时的万能缓冲
- **话术：** “这个问题角度确实很刁钻，我刚才在下面也算了半天没算通，给我两分钟我再核算确认一下回复大家。”
### 3.2 观点冲突时的危险发言
- **话术：** “我理解老王的出发点是为了转化率，非常有道理。我从研发角度补充一个‘不一定对’的视角，咱们看看怎么结合一下。”

## 四、 极限拉扯：不给结论就不散会？
### 4.1 脱身话术
- **场景：** 临近下班被拉着开无休止的会。
- **干预动作：** 站起身在白板写总结：“刚大家说的很棒，为了不浪费时间，咱快速过一遍：待办1是...待办2是...责任人分别是...”
- **效果：** 强行收拢话题，展现控场力。
