# 体面退场白皮书：撕破脸是事故，好聚好散是故事

## 一、 冲突调解：当火药味燃起时
### 1.1 跨部门冲突破冰
- **场景：** 两个部门互相指责，会议陷入僵局。
- **高情商破局：** “大家先停一下喝口水。吵了二十分钟其实咱们目标是一致的，都是怕上线崩盘。现在这样，技术这边列出风险清单，运营这边列出必须上线的理由清单，咱们逐条过，能规避的就过，规避不了的共同找领导背锅。”
### 1.2 同事情绪爆发安抚
- **话术：** “知道你最近压力大，换我我也急。先别气，你先把最难的点扔出来，看看我这有什么资源能帮你托底的。”

## 二、 离职前兆：如何不声不响做准备
### 2.1 离职信号管理
- **禁语：** 对任何同事透露离职意向。
- **动作：** 整理工作SOP云端本地双备份；清理公司电脑个人隐私；悄无声息更新简历。

## 三、 史上最稳离职话术（软钉子版）
### 3.1 提出离职那一刻
- **公式：** 感谢过去 + 个人发展规划不符 + 坚决态度。
- **现成话术：** “老大，承蒙照顾这几年学了不少东西。不过出于个人职业规划的原因，我后续想往XX方向深耕，所以提出了离职，感谢栽培。”
### 3.2 面对领导画饼挽留
- **场景：** “正打算给你涨薪呢，留下来晋升机会很大。”
- **高情商回应：** “谢谢老大这份认可，真的特别感动。但这个决定我是深思熟虑过的，确实是想换个赛道看看，如果现在因为加薪留下，我对不起您的这份诚意。交接的事您放心，我会站好最后一班岗。”
- **核心要点：** 温和而坚定，拒绝口子一点不留。

## 四、 离职交接与背调预埋
### 4.1 交接的格局
- **动作：** 拉群交接，发送《详细交接清单.pdf》，并@领导确认。
- **话术：** “所有文档都在表里了，离职后如果遇到没写到的地方，也欢迎随时问我，24小时内必回。”
### 4.2 防范竞业与背调风险
- **离开前最后话术：** “因为下一家公司的背调流程比较严格，可能近期会麻烦您帮忙留意下电话，先谢谢啦。”
- **潜台词：** 我下一家有背调，别使绊子，有电话请美言。
